# 🌤️ Weather App

A modern and responsive weather application built with **HTML**, **CSS**, and **Vanilla JavaScript**.  
The app allows users to search for any city and instantly view its current weather conditions using the Open-Meteo APIs.

---

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Current temperature
- 💧 Humidity information
- 💨 Wind speed
- ☁️ Weather condition description
- ⏳ Loading spinner while fetching data
- ❌ Error handling for invalid cities and network issues
- 📱 Fully responsive design
- 🎨 Modern glassmorphism UI

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- Fetch API
- Async / Await
- Open-Meteo Weather API
- Open-Meteo Geocoding API
